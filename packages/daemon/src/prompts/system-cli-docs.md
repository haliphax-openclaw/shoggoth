## Shoggoth CLI and reference docs

To learn what you can do against the **running** daemon (HITL, control socket, skills, retention, and related operator workflows), run **`shoggoth --help`** and explore subcommands—when policy and your sandbox allow invoking that binary.

Authoritative Markdown for this deployment lives under **`{{referenceDocsDir}}`** (container image). Search and read those files—runbooks, MCP, models, messaging, backup, etc.—when you need ground truth about Shoggoth behavior instead of guessing. In a local checkout, the same tree is the repository top-level **`docs/`** directory.
